# TIG240 Portfolio

Welcome. This site collects the work of all 14 student project groups in **TIG240 — Data-Driven Organizational Development**, Spring 2026.

## The four challenges

The class is organized around four broad challenges. Each has 3–4 groups working in parallel.

- **[Polarization](challenges/polarization/index.md)** — Hamra · Sarek · Pieljekaise · Tiveden
- **[Heal Internet](challenges/heal-internet/index.md)** — Björnlandet · Stenshuvud · Sonfjället
- **[Eurostack](challenges/eurostack/index.md)** — Garphyttan · Muddus · Töfsingdalen
- **[Job Automation](challenges/job-automation/index.md)** — Abisko · Padjelanta · Skuleskogen · Ängsö

## How this site is built

This is a working seed repository. The architectural decisions that shape it — purpose, schema, coupling, axes of variation — are made by the class itself in **Workshop Session 1** and recorded under [Decisions](decisions/index.md). The individual group pages are added in **Workshop Session 2** by their respective groups.

If you're a student in this course, see the repo's `CONTRIBUTING.md` for how to add or update your group's page.

## Acknowledgements

The architecture of this site draws explicitly on Herbert Simon's *The Sciences of the Artificial* — in particular, his ideas of **near-decomposability** (modules within modules) and **bounded rationality** (satisficing under time pressure). The session in which the class designed it was itself an experiment in near-decomposable group work.
